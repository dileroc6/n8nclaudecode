Reglas de Clases CSS para Kadence

En el panel "Avanzado" de cada bloque Kadence, agrega estas clases CSS:

  • tf-glass-card — para cards con efecto cristal
  • tf-badge — para badges de sección (texto pequeño uppercase)
  • tf-grad-text — para texto con gradiente violet→cyan
  • services-grid, why-grid, feature-grid, etc. — en los Row Layouts para que el responsive.css los pille
