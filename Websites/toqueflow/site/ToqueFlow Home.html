🧱 Estructura de Bloques Kadence

1
Kadence Row Layout — Hero Section Full-width, min-height 100vh, background color #010101. Añade clase CSS "tf-hero". Columna centrada con: Kadence Advanced Text (badge pill), H1 con span gradiente, párrafo de subtítulo, Kadence Button Group (2 botones).
2
Kadence Row Layout — Logos Strip Padding top 40px, border-top. Texto pequeño uppercase + fila de Kadence Buttons con estilo "outline pill" para logos de clientes.
3
Kadence Row Layout — Stats Bar (4 columnas) Fondo #0a0a12, padding 60px. Cada columna: H3 con clase "tf-grad-text" (número), párrafo uppercase pequeño (label). Separar con border-right via CSS.
4
Kadence Row Layout — Servicios (3 columnas iguales) Fondo #010101, padding 120px 80px. Encabezado de sección arriba (badge + H2 + subtítulo centrados). Cada columna con clase "tf-glass-card": ícono (div con clase "tf-icon-circle"), H3, párrafo, Kadence Button outline.
5
Kadence Row Layout — Por Qué Nosotros (2 columnas) Fondo #0a0a12. Columna izquierda: badge, H2 con span gradiente, párrafo, métricas en fila. Columna derecha: 3 × Kadence Icon Box con ícono, título y descripción.
6
Kadence Row Layout — 3 Pasos (4 columnas) Fondo #0d0d1a. Encabezado centrado arriba. Cada columna "tf-glass-card": Kadence Icon con fondo gradiente circular, H4, párrafo, botón CTA primario.
7
Kadence Row Layout — CTA Final Fondo #0a0a12. Columna única centrada con clase "tf-glass-card", max-width 700px. H2 con span gradiente, párrafo, Kadence Button Group.
8
Kadence Row Layout — Footer Fondo #010101, border-top. 4 columnas: Brand (logo + texto + redes), Servicios (links), Empresa (links), Contacto. Footer bottom con copyright.
