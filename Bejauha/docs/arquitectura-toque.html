La plataforma Toque (frontend HTML) vive en Hostinger, misma cuenta del VPS. La fuente de verdad es Supabase (BD + API + Auth) — que además es el contrato natural con n8n: su API REST + los webhooks de base de datos.

Las 3 reglas de oro

1. La plataforma es dueña de los datos

La base de datos de Toque es la única fuente de verdad. Ni n8n ni un Sheet guardan el estado del negocio.

2. n8n es un worker sin estado

Solo integra y ejecuta (WhatsApp, IA, crons, envíos). La plataforma lo dispara; él reporta de vuelta. No guarda reglas ni datos críticos.

3. Se hablan por un contrato

API + webhooks entre plataforma y n8n. Nada de bases compartidas ni lógica duplicada en dos lados.

Por qué escala: el mismo motor de n8n sirve a todos los clientes de Toque. Sumar un cliente nuevo es configurarlo en la plataforma (tenant, campañas, plantillas), no rehacer flujos.
